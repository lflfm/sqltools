Working with GIT and SVN

SQLTools does not support any source control system directly but there is a very convenient way to work with GIT and SVN and possible with other source controls.

SQLTools integrates Window Shell menu in order to provide access to TortoiseGIT and TortoiseSVN. TortoiseSVN commands are available from two places:

  • The context menu of the File panel
  • The alternative context menu of the editor, activated by Right Mouse Button Click with <Shift> pressed
