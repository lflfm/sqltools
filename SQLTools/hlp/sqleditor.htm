Executing SQL Statements

In order to execute a sql statement just place the cursor anywhere between the first and last line of the statement and press F10 or Ctrl+Enter.

F5 executes all of the SQL statements in the editor window.

You also can select a part of the text and do the same above.

The executed statements and captured output of dbms_output are logged in the output tab.

If the current statement is a query then its result is fetched in the data grid.

TIPS:

  • F10 executes a quick query from a table/view under the cursor w/o writing select statement.
  • Ctrl+F10 runs "SELECT Count(*) " from a table/view under the cursor
