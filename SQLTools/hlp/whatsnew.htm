What's new in 2.0

Unicode support

The program partially supports Unicode/UTF now. The editor works with European languages w/o any issues but eastern characters are not supported mainly because the Windows monospaced fonts are actually not fixed-width for those code pages. The data grid, on other hand, should handle any texts.

New docking window framework

The control panels like "File Panel", "History", "Open Files", "Object Viewer" can be docked more easily, as well as stacked in tabs. Additionally panels can "Auto hide" in the docking mode. The window tab control "Workbook" is more functional and nicer than in the version 1.9.

Re-implemented "Find In Files"

The external grep program was replaced with the internal implementation which is faster and more functional. See Find in Files for more details.
