Find in Files

Find in Files allows you to search a specified set of files. The matches found and actions taken are listed in the Find Results window

Find what

To search for a new text string or expression, specify it in the box. Choose the adjacent Expression Builder button if you want to use one or more regular expressions in your search string.

File masks

This comma delimited list indicates the masks (with wildcards ? and *) of files to search through in the Look in directories. If this field is blank, all of the files in the Look in directories will be searched.

Look in

Specify the folder or multiple folders where the searching takes place. This is semi-colon delimited list. Click the Browse (...) button to add or modify any component of the list.

You can combine the following placeholders with explicitly entered pathes:

Placeholders Description
<CURRENT_DIR> or <CD> The directory of the active file. If the file is new then te directory of the open workspace will be used otherwise it is undefined.
<WORKSTACE_DIR> or <WD> The directory of the active workspace.
<OPEN_DIRS> or <OD> The directories of all open files.
<BACKUP_DIR> or <BD> The program backup folder.

Math whole word only

When selected, the Find Results windows will only return whole word matches.

Match case

When selected, a Find Results search will be case-sensitive.

Regular Expressions

If this check box is selected, you can use special notations to define patterns of text to match in the Find what text box.

Backslash expressions

Check this option if you want to use the followin unprintable characters in Find what:
    \a     Bell (alert)
    \b     Backspace
    \f     Formfeed
    \t     Horizontal tab
    \v     Vertical tab

Share search context with editor search (F3)

Check this option to share Find in File options with Edior Search.

Look in subfolders

Search also sub-folders of the specified folder(s).

Search in open files

Check this option if you want to search matching open files rather than their saved versions.

Collapsed search results

Check this option if you want to see only file names in Find Results. You will be able to expand a file item and see matching lines.
